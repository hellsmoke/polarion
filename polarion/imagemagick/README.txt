==========================================
Polarion OLE Converter - ImageMagick Setup
==========================================

Installation
============ 

Download Image Magick from http://www.imagemagick.org/script/binary-releases.php,
e.g. distribution ImageMagick-6.8.0-Q16-windows-X64-static.exe.

Install Image Magick into "Polarion Install Dir"/polarion/imagemagick. Path to 
file 'convert.exe' should look like C:\Polarion\polarion\imagemagick\convert.exe.

Follow description for OLE Converter in polarion.properties for 
Single instance or _common.properties for Multi instance
