Element <enum>
The enumeration config. There are three ways how an enumeration can be configured 1) By providing 'factory' - a service which implements IEnumerationFactory, which ends up in completely custom enumeration implementation. 2) By providing 'optionClass' (optional) and 'configFilePath' - this will create enumeration based on configuration in the repository. The configFilePath is relative to the context root. In this case also the controlKey attribute can be provided, specifying the PObject key, which controlls the set of available options. 3) By providing 'optionClass' (optional) and bunch of 'option's - this will create constant enumeration.
